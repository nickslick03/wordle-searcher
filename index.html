    About

    Wordle searcher allows you to take your current Wordle progress and search for all possible results.
    • Type in a word to search against (tap the input box if you're on mobile).
    • Click the letter tiles to change the color of the title.
    • Use the plus (+) and minus (-) buttons to add/remove words to search against; click a word row to select it.
    • Click the search button to show a list of all possible words based on the provided Wordle progress.
